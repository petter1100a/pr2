# Practical 7: Structural Damage Classification

## Theory
This practical applies the **Immune System Analogy** to engineering.
- **Self:** The normal, healthy state of a structure.
- **Non-Self:** Damaged or abnormal states.
- **Detectors:** Algorithms trained to recognize "Non-Self" patterns (damage) using distance functions (like Euclidean distance).

## How to Run
Run the notebook to see how the system classifies input as 0 (No Damage) or 1 (Damage).
