# Practical 6: Clonal Selection Algorithm (AIS)

## Theory
The **Clonal Selection Algorithm** is a class of Artificial Immune Systems (AIS) inspired by how the biological immune system produces antibodies.

### Process:
1. **Selection:** Best performing individuals (antibodies) are selected.
2. **Cloning:** Selected individuals are copied.
3. **Hyper-mutation:** Clones are mutated (randomly changed) to find better solutions.
4. **Affinity:** The "Fitness" of the antibody is measured against the objective (antigen).

## How to Run
Run the Notebook to see the "Best Affinity" improve over 50 generations.
