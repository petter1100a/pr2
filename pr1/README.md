# Practical 1: XML-RPC Factorial Service

## Theory
**XML-RPC** (Extensible Markup Language Remote Procedure Call) is a protocol that uses XML to encode its calls and HTTP as a transport mechanism. 
- It allows a client to call a function on a remote server as if it were a local function.
- In this practical, the server hosts a `factorial(n)` function, and the client invokes it over the network.

## How to Run
1. Run `server (1).py` first.
2. Run `client (1).py` to request the factorial value.
