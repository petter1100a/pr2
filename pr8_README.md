# Practical 8: Genetic Algorithms with DEAP

## Theory
**DEAP** (Distributed Evolutionary Algorithms in Python) is a framework for evolutionary computation.

### GA Steps:
1. **Population:** A group of potential solutions.
2. **Selection:** Keeping the "fittest" solutions (Tournament selection).
3. **Crossover:** Combining parts of two solutions (Two-point crossover).
4. **Mutation:** Randomly flipping bits (Flip-bit mutation).

## How to Run
Ensure `deap` is installed (`pip install deap`). Run the notebook to see the "OneMax" fitness evolve.
