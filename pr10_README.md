# Practical 10: Ant Colony Optimization (ACO)

## Theory
**ACO** is a metaheuristic inspired by the foraging behavior of ants to solve combinatorial problems like the Traveling Salesman Problem (TSP).

### Mechanism:
1. **Pheromone Trail:** Ants drop chemicals on the ground. Shorter paths get more pheromone.
2. **Evaporation:** Pheromones fade over time to prevent getting stuck in local optima.
3. **Probabilistic Choice:** Ants choose the next city based on pheromone strength and path distance.
